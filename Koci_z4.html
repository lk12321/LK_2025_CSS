Metody podłączania CSS do strony WWW

1) Zewnętrzny arkusz stylów

<head>
<link rel="stylesheet" type="text/css" href="style.css" />
</head>

2) Styl lokalny (bezpośrednio w elemencie)

<element style="cecha1: wartość1; cecha2: wartość2">...

3) Wewnętrzny arkusz stylów

<head>
<style type="text/css">
selektor1 { cecha: wartość; cecha2: wartość2; }
selektor2 { cecha: wartość; cecha2: wartość2; }
</style>
</head>

4) Alternatywny arkusz stylów

<head>
<link rel="stylesheet" type="text/css" href="style.css" title="Domyślny styl" />
<link rel="alternate stylesheet" type="text/css" href="style1.css" title="Styl 1" />
<link rel="alternate stylesheet" type="text/css" href="style2.css" title="Styl 2" />
</head>

5) Importowanie arkusza stylów

<style type="text/css">
@import url("ścieżka do zewnętrznego pliku stylów");
</style>
