Budowa selektora

  1. 1. Jakie części składają się na selektor i co oznaczają?

    Selektor wskazuje, do którego elementu HTML odnosi się dany styl. Może być to tag HTML, identyfikator lub klasa.

    Deklaracja to zbiór właściwości i ich wartości, które są przypisywane do określonego elementu HTML, aby nadać mu odpowiedni wygląd.

  2. 2. Jakie znaczenie mają nawiasy klamrowe {...} w selektorze?

    Nawiasy klamrowe {...} są używane do otaczania deklaracji CSS w selektorze. Wewnątrz tych nawiasów umieszczamy właściwości, które będą stosowane do wybranego elementu HTML. Przykład:

    p {
    color: red;
    font-size: 16px;
    }

    W tym przypadku nawiasy klamrowe obejmują deklaracje stylów dla elementu p.

  3. 3. Czym są cechy w CSS?

    Cechy CSS to właściwości, które możemy przypisać do elementu. Każda cecha ma przypisane wartości, które mogą zostać zastosowane. Na przykład cecha text-align odpowiada za wyrównanie tekstu i może przyjąć wartości: left, right, center, justify. Cechy są podstawą określania wyglądu elementów na stronie.

  4. 4. Co to są wartości w CSS?

    Wartości to konkretne parametry przypisane do cech. Każda cecha ma określony zestaw wartości, które mogą być użyte w danej deklaracji. Na przykład dla cechy text-align dostępne wartości to: left, right, center, justify. Przypisanie niewłaściwej wartości, np. przypisanie koloru do cechy text-align, będzie błędem.

  5. 5. Kiedy należy używać średnika (;) w CSS i kiedy jest on zbędny?

    Średnik (;) jest używany do oddzielania poszczególnych właściwości w deklaracji CSS. Jeśli mamy więcej niż jedną właściwość, każdą należy zakończyć średnikiem, np.:

    p {
    color: red;
    font-size: 16px;
    }

    Średnik jest nieobowiązkowy po ostatniej właściwości w deklaracji, jednak zaleca się jego stosowanie po każdej właściwości, aby zachować porządek i poprawność składniową.
