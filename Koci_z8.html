Wewnętrzny arkusz stylów

  1. 1. Gdzie umieszczamy wewnętrzny arkusz stylów?

    Wewnętrzny arkusz stylów jest umieszczany w sekcji <head> dokumentu HTML, pomiędzy znacznikami <style> i </style>. Wszystkie style są zapisane wewnątrz tych znaczników.

  2. 2. Jak wygląda składnia teoretyczna stylu lokalnego (z użyciem ramki)?

    <head>
    <style type="text/css">
    selektor1 { cecha: wartość; }
    selektor2 { cecha2: wartość2; }
    </style>
    </head>

  3. 3. Jakie rodzaje selektorów możemy używać?

    Wewnętrzny arkusz stylów umożliwia stosowanie różnych selektorów HTML, takich jak:

    • h1 - nagłówek poziomu 1
    • div - element blokowy
    • p - akapit
    • span - element inline
    • #identyfikator - selektor identyfikatora
    • .klasa - selektor klasy
    Selektory definiują, które elementy HTML będą miały przypisane określone style.

  4. 4. Jakie cechy i wartości mogą być używane w stylach?

    Cecha to właściwość elementu, którą chcemy zmienić. Przykłady cech to:

    • background-color - kolor tła
    • font-size - rozmiar czcionki
    • color - kolor tekstu
    • margin - margines
    Wartości to konkretne ustawienia, które przypisujemy do cech. Przykłady wartości to:
    • #FFFF00 - dla background-color
    • 10px, 1em - dla font-size
    • purple, #800080 - dla color
    • 0, 10px - dla margin
    Deklaracje stylów są tworzone, łącząc cechy i ich wartości, aby określić wygląd poszczególnych elementów.
