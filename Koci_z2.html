Definicja CSS - Kaskadowe Arkusze Stylów

CSS - Cascading Style Sheets (z ang. "Kaskadowe Arkusze Stylów") to specjalny język stworzony w celu elastycznego zarządzania wyglądem elementów na stronach WWW. CSS pozwala na definiowanie, jak elementy HTML mają być wyświetlane, w tym układ, kolory, czcionki i inne style.
